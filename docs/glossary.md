# Glossary

- Control Center: Admin UI and API for platform operations.
- Project Manifest: metadata file that attaches an external app project to the platform.
- Provider Profile: environment/provider behavior selector.
- Evidence: non-sensitive proof generated by scripts or providers.
- Live Proof: evidence from real external systems.
- Production Go/No-Go: hard release/deploy readiness decision.
- Release Evidence: proof for images, SBOM, provenance and rollback.
- SBOM: software bill of materials.
- Provenance: statement describing how an artifact was built.
- GitHub/Sigstore Attestation: GitHub-signed artifact provenance.
- RPO: recovery point objective.
- RTO: recovery time objective.
- WAF: web application firewall.
- Off-site Restore: restore test from storage outside the host.
- Digest-pinned image: container image referenced by immutable digest.
